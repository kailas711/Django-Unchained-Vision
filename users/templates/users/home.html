This Project was done when i was in 3rd year using Django, Ultralytisc YOLOv8n and HTML/CSS/Javascript templates. The goal behind this work is to make a intruder alert system that allows people to "draw" on screen to mark regions in a place that are prohibited. The system detects people and raise alarm sound. Mouse movements are recorded via Javascript to mark the regions. The app takes in IP camera stream as input. For this pourpose i've used "Ip Webcam App" from playstore.
